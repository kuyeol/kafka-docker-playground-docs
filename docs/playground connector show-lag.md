# playground connector show-lag

🐢 Show lag of sink connector

## Usage

```bash
playground connector show-lag [OPTIONS]
```

## Options

#### *--connector, -c CONNECTOR*

🔗 Connector name  
  
🎓 Tip: If not specified, the command will apply to all connectors

#### *--wait-for-zero-lag*

😴 Wait until lag becomes 0
