```bash

```

## Examples

```bash
playground get-jmx-metrics --component connect
```

```bash
playground get-jmx-metrics --component connect --domain "kafka.connect kafka.consumer kafka.producer"
```

```bash
playground get-jmx-metrics -c broker
```
