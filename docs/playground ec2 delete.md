# playground ec2 delete

❌ Delete an EC2 instance created with Cloud Formation

## Usage

```bash
playground ec2 delete [OPTIONS]
```

## Options

#### *--instance, -i INSTANCE*

🌀 EC2 instance cloudformation (need to use completion to get all required details)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
