# playground topic delete

❌ Delete topic and associated schema/subject if applicable

## Usage

```bash
playground topic delete [OPTIONS]
```

## Options

#### *--verbose, -v*

🐞 Show command being ran.

#### *--topic, -t TOPIC*

🗳 Topic name

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *--skip-delete-schema*

🔰 Do not delete subject/schema
