# playground fully-managed-connector pause

⏸️  Pause connector

## Usage

```bash
playground fully-managed-connector pause [OPTIONS]
```

## Options

#### *--verbose, -v*

🐞 Show command being ran.

#### *--connector, -c CONNECTOR*

🔗 Connector name  
  
🎓 Tip: If not specified, the command will apply to all connectors
