# playground ccloud-connector resume

⏯️  Resume connector

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | unpause

## Usage

```bash
playground ccloud-connector resume [OPTIONS]
```

## Options

#### *--verbose, -v*

🐞 Show command being ran.

#### *--connector, -c CONNECTOR*

🔗 Connector name  
  
🎓 Tip: If not specified, the command will apply to all connectors
